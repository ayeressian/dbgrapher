

# DBGrapher
A database schema designing tool that runs in browser.

## To run
1. Clone the repository.
2. Run "npm i".
3. Create a file with the name env.json in the root directory with the following content.
  ```JSON
  {
    "googleDrive": {
      "developerKey": "",
      "clientId": "",
      "appId": ""
    }
  }
  ```
4. Run "npm start".
